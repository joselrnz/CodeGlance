# Task Manager (example)

A tiny example app used to demonstrate `codeglance`.

- `app/main.py` — entry point
- `app/routes.py` — request routing
- `app/models.py` — `Task` and `User` domain models
- `app/db.py` — in-memory storage
- `app/config.py` — configuration

Run: `codeglance examples/taskman`
